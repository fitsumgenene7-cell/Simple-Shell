## Project Description
OShell is a simplified Unix command-line interpreter that implements core shell features including command execution, built-in commands, variable expansion, and redirection.

## Features Implemented
- [ ] Interactive, batch, and pipe modes
- [ ] Command parsing with operators (;, &&, ||, &, >, #)
- [ ] Built-in commands (cd, exit, env, setenv, unsetenv, alias, path)
- [ ] Variable expansion ($VAR, $?, $$)
- [ ] PATH-based command execution
- [ ] Redirection (>)
- [ ] Error handling with unified error message
- [ ] Signal handling (Ctrl+C ignored, Ctrl+D exits)
